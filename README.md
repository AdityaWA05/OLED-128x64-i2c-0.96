## 资料描述
我们使用的oled lcd驱动模块IC是旧的输出ic，即sd1306。
<br />
它比最新的（2022）输出驱动器IC，即sd1315昂贵得多。
<br />
由于大多数0.96OLED Lcd已销售或使用，因此驱动器ic为SD1306。驱动ic Sd1315与sd1306兼容
